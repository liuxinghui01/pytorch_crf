# pytorch_crf
pytorch version of conditional random field
条件随机场的pytorch版本，知乎关于数据集和模型推导的说明：https://zhuanlan.zhihu.com/p/546719981

## 文件结构

——data_processor.py #一些数据处理的函数

——main.py #代码运行主函数

——model.py #模型代码

——train.py #训练代码

——valid.py #验证模型代码

## 模型运行
```
main.py
```
